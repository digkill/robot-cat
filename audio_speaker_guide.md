# Подключение динамика к Raspberry Pi Zero 2 W

У Pi Zero 2 W **нет разъёма 3.5 мм** для звука. Ниже варианты от простого к продвинутому.

---

## 1. USB-динамик или USB-звуковая карта (проще всего)

**Что нужно:** колонка или наушники с USB (или USB‑звуковая карта + обычный динамик/наушники).

**Подключение:** вставить в **USB‑порт** Pi Zero 2 W (через переходник micro USB → USB A, если у колонки обычный USB‑штекер).

**В системе:** Raspberry Pi OS обычно сразу видит USB‑аудио. Проверка:

```bash
# Список устройств воспроизведения
aplay -l

# Тест (если есть файл WAV)
aplay /usr/share/sounds/alsa/Front_Center.wav
```

Если устройство не выбрано по умолчанию, укажи его в настройках или в `~/.asoundrc` / `/etc/asound.conf`.

---

## 2. I2S DAC + динамик (качество получше, без USB)

**Что нужно:** плата I2S DAC (например PCM5102, UDA1334A, MAX98357A) и динамик (или маленький усилитель + динамик).

**Подключение к GPIO (BCM):**

| I2S DAC (PCM5102 / UDA1334) | Raspberry Pi Zero 2 W |
|----------------------------|------------------------|
| VIN / 3.3V                 | 3.3V (pin 1 или 17)    |
| GND                        | GND (pin 6, 9, 14…)    |
| BCK / BCLK                 | GPIO 18 (pin 12)       |
| DIN / DOUT / Data          | GPIO 21 (pin 40) *     |
| LRC / WS / LRCLK           | GPIO 19 (pin 35)       |

\* На части плат пин данных подписан DIN или DOUT — это один и тот же сигнал к Pi.

**Включение I2S в системе:**

В `/boot/firmware/config.txt` (или `/boot/config.txt`) добавить:

```ini
# I2S для звуковой платы (раскомментируй одну строку под свою плату)
# dtparam=i2s=on
# dtoverlay=hifiberry-dac
# dtoverlay=googlevoicehat-soundcard
dtoverlay=i2s-mmap
dtoverlay=simple-audio-card,string=imx283
```

Для популярных HAT/DAC часто есть готовый overlay, например для PCM5102:

```ini
dtparam=i2s=on
dtoverlay=hifiberry-dac
```

После правки — перезагрузка: `sudo reboot`.

Дальше снова проверить: `aplay -l` и тестовый WAV.

---

## 3. Bluetooth-колонка или наушники

**Что нужно:** любая Bluetooth‑колонка или наушники.

На Pi:

```bash
# Включить Bluetooth, подключить колонку через меню (иконка Bluetooth в трее)
# или через консоль:
bluetoothctl
# В bluetoothctl: scan on, pair <MAC>, connect <MAC>, trust <MAC>
```

После подключения в настройках звука (или через PulseAudio) выбрать устройство вывода «Bluetooth».

---

## Сводка

| Способ              | Плюсы                    | Минусы              |
|---------------------|--------------------------|----------------------|
| USB-динамик/звук    | Без пайки, сразу работает| Занимает USB‑порт    |
| I2S DAC + динамик   | Хорошее качество, GPIO   | Нужна плата и провода|
| Bluetooth           | Без проводов             | Зависит от Блютуза   |

Для быстрого старта удобнее всего **USB-колонка**. Если напишешь, какой у тебя динамик (USB, просто два провода, или плата с названием), можно расписать подключение под твой вариант пошагово.
